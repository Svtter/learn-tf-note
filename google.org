* Deep learning

** 从机器学习到深度学习
*** softmax
    softmax可以把数值转换为对应在0-1之间的概率。
   
    $S({y_i}) = \frac{e^{y_i}}{\sum_{j}e^{y_{j}}}$
   
    #+BEGIN_SRC python
   
    def softmax(x):
        return np.exp(x) / np.sum(np.exp(x), axis=0)

    #+END_SRC
